# Сайт-портфолио на MkDocs

Данный репозиторий содержит статический сайт, созданный с помощью **MkDocs** и опубликованный через **GitHub Pages**.

## Выбор темы оформления

Для сайта была выбрана тема **Material for MkDocs**, так как она:

- имеет современный и аккуратный внешний вид
- поддерживает удобную иерархическую навигацию
- делает страницы более читаемыми и структурированными

## Публикация

Сайт публикуется через GitHub Pages из каталога `/docs` ветки `main`.
